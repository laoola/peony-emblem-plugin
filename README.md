# peony-emblem-plugin

银河麒麟 Peony 文件管理器安全级别角标插件

## 项目简介

这是一个为银河麒麟系统下的Peony文件管理器开发的安全级别角标插件，能够根据文件名中包含的安全级别关键词自动为文件显示相应的角标图标，帮助用户快速识别文件的安全级别。

## 功能特点

- 支持8种安全级别：公开、普通、内部、秘密、机密、绝密、核心、非密
- 根据文件名中的关键词自动识别安全级别
- 为文件添加对应的角标图标
- 支持中英文关键词识别

## 安全级别对应关系

- 公开 (gongkai) - 蓝色角标
- 普通 (putong) - 绿色角标
- 内部 (neibu) - 黄色角标
- 秘密 (mimi) - 橙色角标
- 机密 (jimi) - 红色角标
- 绝密 (juemi) - 深红色角标
- 核心 (hexin) - 紫色角标
- 非密 (feimi) - 灰色角标

## 项目结构

- `emblemplugin.cpp/h` - 插件的主要实现文件
- `res/` - 角标图标资源目录
- `plugin-iface/` - 插件接口定义文件
- `emblem.pro` - Qt项目文件
- `emblem_plugin.json` - 插件元数据文件
- `emblem_res.qrc` - 资源文件

## 安装说明

1. 确保已安装Qt开发环境和Peony文件管理器开发库
2. 编译插件：

   ```bash
   qmake
   make
   ```
3. 编译后的插件文件将生成在plugins目录下
4. 将插件复制到Peony的插件目录中

## 使用方法

安装插件后重启Peony文件管理器
创建或重命名文件，在文件名中包含相应的安全级别关键词
文件图标将自动显示对应的安全级别角标

## 注意事项

目前插件仅处理文件，不处理文件夹
如果文件名中同时包含多个安全级别关键词，将优先匹配在代码中定义的第一个匹配项
图标资源可以通过资源文件加载或从本地路径加载
